# 📊 Интерактивный конструктор коммерческих предложений (NextBot Automation)

> **GitHub Pages:** [https://yernaribadulla.github.io/Check-List/](https://yernaribadulla.github.io/Check-List/)

Веб-инструмент для демонстрации и динамического расчета стоимости услуг по автоматизации бизнеса и интеграции ИИ-инфраструктуры. Изначально спроектирован для проекта автономной плачущей кофейни в г. Актау.

---

## 🚀 Функционал и особенности

- **Полностью адаптивный интерфейс:** Корректно отображается на любых устройствах (включая учет безопасных зон `viewport-fit=cover` на смартфонах с челками).
- **Динамический расчет стоимости:** Сумма пересчитывается мгновенно "на лету" при выборе или кастомизации этапов внедрения.
- **Интерактивные элементы:** Круговой SVG-индикатор (Gauge), отображающий процент загрузки предложения от полного объема.
- **Умные мастер-тумблеры:** Возможность массового выбора или сброса чекбоксов для целых этапов разработки.
- **Zero-Dependency:** Чистый HTML5, CSS3 (переменные, `@keyframes` анимации) и ванильный JavaScript. Никаких тяжелых внешних библиотек или фреймворков.
- **Интеграция с мессенджерами:** При клике на кнопку «Принять предложение» скрипт автоматически парсит выбранные клиентом модули, формирует красивый текстовый чек-лист с итоговой ценой и генерирует прямую ссылку для отправки в Telegram.
